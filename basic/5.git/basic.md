# **基本編5：Git/GitHub**
## Git編
### 51. gitの実行環境をインストールしなさい

### 52. [spring-boot](https://github.com/spring-projects/spring-boot) のリポジトリを自身のローカル環境にクローン（clone）しなさい

### 53. クローンしたspring-bootの最新コミットに含まれるjavaファイルの数を求めなさい

### 54. 最新コミットに含まれる各ファイルの行数を求め，上位10件のファイル名と行数を求めなさい

### 55. `spring-boot`の各ブランチがどのような用途で使われているかを推測しなさい
gitの **ブランチ** (branch) は`git branch -r`で確認できる．

### 56. `spring-boot` のタグをカウントしなさい
gitには **タグ** と呼ばれる機能があり，特定のコミットに別名を付与することができる．タグを確認するためのコマンド `git tag` を実行し，`spring-boot`のタグ数をカウントしなさい．

### 57. `spring-boot`のタグ一覧を分析し，それぞれのタグ名がどういった意図で作成されているかを推測しなさい

### 58. `spring-boot`の`main`ブランチから辿れるコミットの数を求めなさい
`git log –oneline` コマンドは現在いるブランチから到達可能なコミットのログを1行ずつ全て表示することができる．

### 59. `spring-boot`の各ブランチから辿れるコミットの数を全て求めなさい

### 60. 特定の開発者が行ったコミットを列挙しなさい
`spring-boot`のあるコミットに着目する．このコミットは自由に決めて良いが特に思い着かない場合は`main`ブランチの最新コミットを考える．このコミットのcommitterがコミットした全てのコミットを列挙しなさい．
なお，ここでの全てのコミットとは，`main`ブランチから辿れる全てのコミットとする．
